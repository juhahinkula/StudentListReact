# SpringBootReact
Spring Boot + React.js frontend

How to Launch: 
1. Install npm packages
npm install
2. Run webpack
webpack
3.Run the application
mvnw spring-boot:run

Login by using: user/user or admin/admin

    H2 database
    JPA
    REST
    React.js
    React components: react-skylight (for modals), react-s-alert (for toasts)
